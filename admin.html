Finding Google Form Field Entry IDs
  1. Open your Google Form. Click the eye icon (Preview) in the top-right.
  2. Right-click on any input field and select Inspect.
  3. Find the HTML element for the input (usually a `name` attribute like name="entry.XXXXXX").
  4. Copy and paste that entry name into the corresponding input field above.
